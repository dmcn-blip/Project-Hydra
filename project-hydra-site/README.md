# Project Hydra - Site de Documentação

## Descrição

Site de documentação completo para o projeto **Project Hydra**, uma mão biônica robótica controlada por Arduino desenvolvida como projeto acadêmico para as disciplinas de Introdução à Computação e Sistemas Digitais.

## Características

- **Design Responsivo**: Funciona perfeitamente em desktop, tablet e mobile
- **HTML5 Semântico**: Estrutura bem organizada e acessível
- **CSS Moderno**: Estilos modernos com gradientes, animações e transições suaves
- **JavaScript Interativo**: Funcionalidades como smooth scroll, animações ao entrar na viewport e cópia de código
- **Seções Completas**:
  - Descrição do projeto com motivação, objetivo e contextualização
  - Esquema conceitual do circuito
  - Lista completa de componentes
  - Informações sobre aplicativos e plataformas utilizadas
  - Perfis dos autores com links para LinkedIn e GitHub
  - Código comentado do Arduino
  - Vídeo demonstrativo do YouTube
  - Passo a passo detalhado da montagem
  - Problemas enfrentados e lições aprendidas
  - Seção de download do projeto

## Estrutura de Arquivos

```
project-hydra-site/
├── index.html          # Arquivo HTML principal
├── css/
│   └── style.css       # Estilos CSS
├── js/
│   └── script.js       # Scripts JavaScript
├── images/             # Pasta para imagens (vazia)
├── assets/             # Pasta para assets adicionais (vazia)
└── README.md           # Este arquivo
```

## Como Usar

### Localmente

1. Clone ou baixe os arquivos do projeto
2. Abra o arquivo `index.html` em um navegador web
3. Navegue pelas diferentes seções usando o menu de navegação

### Em um Servidor Web

1. Faça upload de todos os arquivos para seu servidor web
2. Acesse o site através do URL do seu servidor
3. O site funcionará normalmente sem necessidade de configuração adicional

## Navegação

O site possui um menu de navegação fixo no topo com links para:

- **Início**: Seção hero com apresentação do projeto
- **Descrição**: Motivação, objetivo e contextualização
- **Componentes**: Lista de componentes e plataformas utilizadas
- **Autores**: Perfis dos desenvolvedores
- **Código**: Código do Arduino com explicações
- **Vídeo**: Demonstração em vídeo do projeto
- **Download**: Seção para baixar o projeto completo

## Tecnologias Utilizadas

- **HTML5**: Estrutura semântica
- **CSS3**: Estilos modernos com suporte a grid, flexbox e animações
- **JavaScript Vanilla**: Sem dependências externas
- **SVG**: Diagrama do circuito em formato vetorial

## Compatibilidade

O site é compatível com:

- Chrome (versão 90+)
- Firefox (versão 88+)
- Safari (versão 14+)
- Edge (versão 90+)
- Mobile browsers (iOS Safari, Chrome Mobile)

## Funcionalidades JavaScript

### Smooth Scroll
Navegação suave entre seções ao clicar nos links do menu.

### Animações ao Scroll
Elementos animam quando entram na viewport durante o scroll.

### Cópia de Código
Botão para copiar trechos de código para a área de transferência.

### Barra de Progresso
Indicador visual do progresso de scroll na página.

### Rastreamento de Eventos
Sistema básico de rastreamento de eventos para análise.

### Acessibilidade
Melhorias de acessibilidade com atributos ARIA.

## Customização

### Cores
As cores principais estão definidas como variáveis CSS no arquivo `style.css`:

```css
:root {
    --primary-color: #0066cc;
    --secondary-color: #ff6b6b;
    --accent-color: #51cf66;
    /* ... outras cores ... */
}
```

### Conteúdo
Todo o conteúdo está no arquivo `index.html` e pode ser facilmente editado.

### Responsividade
Os breakpoints de responsividade estão definidos em `style.css`:

- Desktop: 1200px
- Tablet: 768px
- Mobile: 480px
